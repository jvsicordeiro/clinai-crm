# Clinai CRM

CRM Financeiro da Clinai. Single-file HTML standalone com React + Supabase.

**Produção:** https://crm.clinai.com.br

## Estrutura

- `index.html` - aplicação completa (React + CSS + JS inline)

## Stack

- React 18 (via CDN)
- Babel standalone (JSX inline)
- Supabase (backend)
- Vercel (deploy)
